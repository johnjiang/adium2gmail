adium2gmail

Converts Adium chatlogs to gmail style emails.

Installation

Run:

$ gem install adium2gmail

Usage

Run:

$ adium2gmail -i --input <path> -o --output <path>

Open the file in Thunderbird to view all the chatlogs

Copy it into your gmail
